# Persona Finder (English)

## Overview

Persona Finder reconstructs the Jungian **Eight Cognitive Functions**  
as a **computational, dynamic cognitive structure**, not a static typology.

It outputs `myself.json`, a structural profile containing:

- cognitive axes  
- functional hierarchy  
- contradiction scores  
- consistency indices  
- interpretive metadata  
- AI-usable cognitive architecture  

---

## Purpose

- Build a computational cognitive architecture  
- **(Planned / partial)** Expose cognition as an API-like profile  
- Evaluate Jungian theory via code  
- Provide transparent alternatives to black-box typology  
- Enable AI agent optimization via cognitive metadata  

---

## Tech Stack

Vanilla JS / Modular architecture / Accessibility / Static hosting ready

---

## Output: `myself.json`

A structured cognitive profile used for:

- research  
- self-analysis  
- AI personalization  
- agent system integration
